Абзацы и форматирование.

3 года занимаюсь програмированием. Это очень увлекательно

Мой любимый язык — Python. Он отлично подходит для начинающих.

Git — лучшая система контроля версий. Она позволяет отслеживать изменения в коде и работать в команде без страха что-то сломать.

Главный навык программиста — отладчик (debugger). Он позволяет выполнять код по шагам и видеть где именно прячется ошибка.

Программирование развивает логическое мышление и учит решать сложные задачи. Этот текст будет обтекать картинку, потому что мы применим CSS.

Тестирование – неотъемлемая часть разработки. Оно помогает находить ошибки до того, как их увидят пользователи. Юнит-тесты проверяют отдельные функции, а интеграционные – как части системы работают вместе.Без автоматических тестов любой проект превращается в "шатающуюся башню из кода".

Как посмотреть изменения в репозитории на GitHub

  1. Открой свой репозиторий на GitHub
  2. Нажми на ссылку с количеством коммитов (например, «3 commits») вверху страницы
  3. Либо нажми на вкладку Commits
  4. Нажми на хеш коммита (например, a1b2c3d), чтобы увидеть, что именно изменилось
